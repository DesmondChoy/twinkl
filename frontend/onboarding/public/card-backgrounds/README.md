# SVBWS Card Background Provenance

These six decorative card backgrounds support the Twinkl onboarding
presentation. They have no value meaning and are not part of scoring.

## Assignment Contract

- A task assigns `memory-atlas-01.jpg` through `memory-atlas-06.jpg` to the six
  randomized display positions.
- The assignment never uses a BWS object key or Schwartz label.
- A card keeps its background while moving between the selection area, Most,
  and Least.
- The same BWS object can receive a different background when it returns in a
  later task because every task has its own randomized card order.

## Generation Provenance

The source images were generated on 2026-07-20 with OpenAI's built-in image
generation tool. No seed or model identifier was exposed by that interface.
The generated 1086 × 1448 PNG files were resized to 600 × 800 JPEG files for
delivery. The app applies the same saturation, contrast, brightness, and text
veil to every image.

Every prompt shared this specification:

> One abstract “memory atlas” portrait card background for a private
> values-reflection onboarding flow. Use tactile editorial illustration with
> translucent vellum layers, soft ink washes, subtle paper grain, and faint
> cartographic linework. Keep the lower third calm for overlay text. Make it
> quiet, humane, introspective, warm, and inviting. Use ink navy, mist blue,
> soft indigo, muted coral, dusk gold, and cloud white. Match luminance,
> contrast, complexity, and color balance across the series. Include no text,
> people, faces, hands, animals, plants, landscapes, buildings, recognizable
> objects, icons, compass symbols, hearts, crowns, shields, flags, religious
> imagery, money, trophies, or literal stars.

The six composition prompts varied only the abstract structure:

1. crossing translucent oval currents;
2. offset horizontal bands and shallow seams;
3. overlapping diagonal vellum planes;
4. open concentric arcs cropped by the edges;
5. vertically meandering ribbon-like fields; and
6. evenly scattered translucent pebble-like fields.

The assets were reviewed for semantic imagery and obvious focal symbols before
integration. This was a product review, not evidence that the images have no
priming effect.
