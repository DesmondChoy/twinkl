Twinkl End-to-End Pipeline

From synthetic data generation and judge supervision to trained Critic checkpoints and runtime weekly feedback.

Offline Data Generation

Persona Generator

LLM creates longitudinal journal trajectories and optional nudge-response turns.

Text Encoder

Frozen sentence encoder converts entries into dense embeddings.

Judge

LLM-as-Judge produces per-dimension alignment labels and rationales.

State Construction

Builds student-visible state vectors from embeddings, timing, and value-profile context.

Training Dataset

State-target pairs are consolidated into the supervised dataset used to train the Critic.

Supervised Training

Critic Training

Ordinal MLP variants are trained, compared, and selected under the corrected holdout regime.

Selected Checkpoint

The active frontier checkpoint is persisted for demo-time inference and weekly review runs.

Runtime Inference

User Journal Entry

Live or cached journal text enters the runtime path.

Text Encoder

The saved runtime stack rebuilds entry embeddings for Critic inference.

Critic Inference

Per-dimension means and uncertainties are produced with MC Dropout.

Weekly Routing

Signals are aggregated into crash/rut-style weekly detections with uncertainty gating.

Coach Digest

Evidence-grounded artifacts are prepared for review, explanation, and future user delivery.

Selected checkpoint weights feed runtime inference.
