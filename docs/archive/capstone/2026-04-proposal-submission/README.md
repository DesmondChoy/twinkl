# April 2026 Project Proposal Submission

This directory is the immutable snapshot of the April 2026 project proposal
that was submitted for the capstone milestone.

## Preservation rule

Do not edit, regenerate, or refresh the files in this directory to match the
current repository. They record what was submitted at that milestone, including
the terminology, metrics, diagrams, and implementation status that were current
then.

If the project needs a current-state report, create a new artifact outside this
archive with a new name and date. Do not reuse the submitted filenames for a
later revision.

## Contents

- `April_Project_Proposal.pdf` is the submitted rendered proposal.
- `April_Project_Proposal.md` is its Quarto source.
- `April_Project_Proposal_slides.md` is the associated slide source.
- The PNG files are the images referenced by those Markdown sources.
- `image_sources/` preserves the HTML sources for the generated proposal
  figures.

The entire bundle was restored to its pre-archive Git content and then moved
here without content edits on 2026-07-10. Byte-for-byte identity was verified
against the original tracked paths.
